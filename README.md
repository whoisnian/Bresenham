# Bresenham

### 环境
ArchLinux  
Breeze Dark Theme [KDE]  
Qt Creator 4.7.1 based on Qt 5.11.2  

### 操作
* 左键单击按钮进行绘图，状态栏会显示平均耗时，其中：  
  * “直接计算”为通过函数式求坐标进行描点绘图；  
  * “算法优化”为利用中点Bresenham算法递推误差项进行描点绘图。  
* 在绘图区域移动鼠标滚轮，对图像进行放大或缩小。  
* 在绘图区域按下鼠标左键进行拖动，调整图像显示位置。  
* 在绘图区域按下鼠标右键，重置绘图区域。
